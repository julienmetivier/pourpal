# PourPal - Next Features

## High Priority

1. **Order Status Workflow**
   - Add statuses: pending → preparing → ready → completed
   - Allow bartenders to update order status as they work through them
   - Visual indicators in Orders list

2. **Order Filtering/Search**
   - Filter orders by status (pending, done, rejected)
   - Search orders by client name
   - Sort by time (newest/oldest first)

3. **Order Notes/Special Instructions**
   - Add notes field when placing orders (e.g., "no ice", "extra lime")
   - Display notes prominently in Orders list

4. **Order Cancellation**
   - Allow canceling orders (different from rejection - for customer requests)
   - Only allow cancellation for pending orders

## Medium Priority

5. **Analytics/Dashboard**
   - Most popular drinks
   - Orders per day/hour
   - Revenue tracking (if you add prices)
   - Simple charts/graphs

6. **Drink Categories Filtering**
   - Filter drinks by category (beer, wine, cocktail) in order page
   - Quick category tabs

7. **Batch Operations**
   - Mark multiple orders as "ready" at once
   - Bulk status updates

8. **Order Timestamps & Wait Times**
   - Show how long orders have been pending
   - Estimated completion time

## Nice to Have

9. **Client Favorites/History**
   - Remember frequent customers
   - Quick reorder from history

10. **Order Editing**
    - Edit orders before they're processed
    - Change drink or client name

11. **Notifications**
    - Sound/visual alerts for new orders
    - Badge count for pending orders

12. **Daily Reports**
    - Summary at end of day
    - Export to CSV/PDF

13. **Multi-Location Support**
    - If you have multiple bars
    - Location-specific settings
